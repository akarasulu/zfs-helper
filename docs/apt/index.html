GH-Repos APT Repository

This is an APT repository hosted on GitHub Pages for GH-Repos.

Quick Setup

# Download and run setup script
curl -fsSL https://akarasulu.github.io/gh-repos/apt/repo-setup.sh | bash

# Or manual setup:
curl -fsSL https://akarasulu.github.io/gh-repos/apt/apt-repo-pubkey.asc | sudo tee /etc/apt/trusted.gpg.d/gh-repos.asc > /dev/null
echo "deb https://akarasulu.github.io/gh-repos/apt stable main" | sudo tee /etc/apt/sources.list.d/gh-repos.list
sudo apt update

Available Packages

dev-tools (1.5.2)

Collection of development utility scripts A comprehensive set of command-line tools for developers including project scaffolding, git helpers, and build utilities. . This package demonstrates: - Configuration file management with conffiles - Dependencies and suggestions - Shared data files installation - Multiple utility scripts in one package

sudo apt install dev-tools

hello-world (1.0.0)

Simple hello world utility A basic command-line utility that demonstrates a minimal Debian package structure. . This package contains a simple script that prints a greeting message to the terminal.

sudo apt install hello-world

mock-monitor (2.1.3)

Mock system monitoring service A demonstration systemd service that simulates system monitoring functionality. . This package shows how to create packages that: - Install systemd service files - Handle service lifecycle in maintainer scripts - Create log directories and manage permissions - Integrate with system service management

sudo apt install mock-monitor

sys-info (3.2.1)

Advanced system information utility A fast, efficient system information tool written in C that provides detailed hardware and software information. . This package demonstrates: - Architecture-specific binaries (amd64) - Compiled C programs with dependencies - Manual pages installation - Build process with compilation

sudo apt install sys-info
