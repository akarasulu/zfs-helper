ZFS Helper APT Repository

This is an APT repository hosted on GitHub Pages for ZFS Helper.

Quick Setup

# Download and run setup script
curl -fsSL https://akarasulu.github.io/zfs-helper/apt/repo-setup.sh | bash

# Or manual setup:
curl -fsSL https://akarasulu.github.io/zfs-helper/apt/apt-repo-pubkey.asc | sudo tee /etc/apt/trusted.gpg.d/zfs-helper.asc > /dev/null
echo "deb https://akarasulu.github.io/zfs-helper/apt stable main" | sudo tee /etc/apt/sources.list.d/zfs-helper.list
sudo apt update

zfs-helper-client (1.0.0)

Client tools for ZFS delegation helper Client-side utilities for the zfs-helper system that allows unprivileged systemd user-scoped services to perform secure ZFS operations. . This package contains the zfs-helperctl command-line tool that communicates with the zfs-helper daemon to request ZFS operations, along with example user service templates and comprehensive documentation. . Install this package on systems where users need to perform ZFS operations through the zfs-helper delegation system.

sudo apt install zfs-helper-client

zfs-helper (1.0.0)

ZFS delegation helper daemon for secure unprivileged ZFS operations A Linux OpenZFS delegation helper service that allows unprivileged systemd user-scoped services to securely request ZFS operations (mount, snapshot, rollback, create, destroy, rename, setprop, and share) on specific datasets and snapshots via a privileged socket-activated helper daemon. . The privileged gateway service securely overcomes delegation limitations resulting from ZFS ports to Linux. It follows a fine-grained user, operation, and delegate service policy with comprehensive audit logging. . This package contains the core daemon and systemd units that run with root privileges to perform ZFS operations on behalf of authorized unprivileged services.

sudo apt install zfs-helper
